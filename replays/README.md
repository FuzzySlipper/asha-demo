# Runtime Replay Evidence

`generated-tunnel-playable-loop.json` is a demo-owned evidence artifact captured from public RuntimeSession telemetry and replay readouts. It records the generated-tunnel seed, native-provider status, replay hashes, and a meaningful loop: collision-constrained movement, enemy attack/player death, typed restart, and primary-fire enemy defeat.

Run `npm run capture:replay` after `npm run build` to refresh it. `npm test` captures and validates the artifact as part of the demo checks. The artifact is observational evidence, not demo-local replay or gameplay authority.
