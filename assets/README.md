# ASHA Demo Assets

The animated character serves the committed Kenney GLB through the public
`@asha/renderer-host` resource manifest. The runtime loads the hash-pinned GLB
from this project asset root; it does not depend on `/home/stash` or engine
fixture paths.

`mesh-animation/kenney-retro-character-medium.manifest.json` records the stable
asset id, SHA-256 content hash, named clips, and license URL. The source asset
is CC0 content created and distributed by Kenney; see the adjacent license
file.

Generated assets, caches, and publish output belong under `dist/`, not in this
source asset root.
