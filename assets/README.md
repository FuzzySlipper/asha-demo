# ASHA Demo Assets

The animated character serves the committed Kenney GLB through the typed stored
presentation catalog. Rust admits its asset identity, source path, content
hash, named clips, and license path before TypeScript derives the public
`@asha/renderer-host` resource descriptor. The runtime does not depend on a
second renderer-only manifest, `/home/stash`, or engine fixture paths.

The audio pulse and particle sprite under `presentation/` are ordinary
manifest-authorized resources referenced by that same stored catalog. The WAV
generation script is an offline source-asset tool; normal build and play never
manufacture replacement bytes. The character source is CC0 content created and
distributed by Kenney; see the adjacent license file.

Generated assets, caches, and publish output belong under `dist/`, not in this
source asset root.
