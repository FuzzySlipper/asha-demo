# ASHA Demo agent guidance

`asha-demo` is the human-facing demonstration repo for ASHA capabilities.

Use public ASHA package roots only. If a demo needs a missing camera, rendering, runtime, asset, or interaction capability, add or request the public ASHA surface rather than recreating engine concepts locally.

Keep synthetic proof harnesses and evidence factories in `asha-testing`; this repo should stay focused on demos that a human can run, inspect, and understand.
